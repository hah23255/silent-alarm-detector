# Silent Alarm Detector

The Silent Alarm Detector project aims to provide a reliable solution for detecting alarms without intrusive sounds. This ensures that you can be alerted in various situations without disturbing others.

## Features
- **Real-Time Detection:** Monitor alarms instantly.
- **User-Friendly Interface:** Easy to use for anyone.

## Installation
To install the project, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/hah23255/silent-alarm-detector.git
   ```
2. Navigate into the directory:
   ```bash
   cd silent-alarm-detector
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
After installation, you can run the project using:
```bash
npm start
```
